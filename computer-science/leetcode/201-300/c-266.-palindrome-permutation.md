# \*C 266. Palindrome Permutation

## Description

Given a string, determine if a permutation of the string could form a palindrome.

**Example 1:**

```text
Input: "code"
Output: false
```

**Example 2:**

```text
Input: "aab"
Output: true
```

**Example 3:**

```text
Input: "carerac"
Output: true
```
